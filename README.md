# Employee Performance Dashboard

## Overview

This project is an interactive HR analytics dashboard developed using **Power BI** and **Microsoft Excel**. It provides insights into employee performance, attendance, productivity, payroll, and workforce distribution through interactive dashboards and visual reports.

## Objectives

- Monitor workforce performance across departments.
- Track key HR metrics using interactive dashboards.
- Analyze employee productivity and attendance.
- Support data-driven business decision-making.

## Tools Used

- Power BI
- Microsoft Excel
- Power Query
- Basic DAX
- Data Visualization

## Dashboard Pages

### Executive Overview
- Total Employees
- Monthly Payroll
- Performance Distribution
- Department Distribution
- Executive KPIs

### Performance Analysis
- Attendance vs. Productivity
- Project Completion by Department
- Performance Score Analysis

### Efficiency & Growth
- Working Hours Analysis
- Productivity Trends
- Talent Distribution

### Employee Directory
- Employee Information
- Department and Job Title Filters
- Performance Status
- Salary Details

- Interactive dashboard pages

- Charts and visualizations
